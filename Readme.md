## Getting Started

Clone the repo:

```bash
git clone https://github.com/PSPDFKit/pspdfkit-web-example-next.js.git
cd pspdfkit-web-example-next.js
```

Install the project dependencies:

yarn:

```bash
yarn
```

npm:

```shell script
npm run install
```
